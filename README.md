Google Machine Learning Crash Course (MLCC)

This repository contains my notes, assignments, hands-on labs, quizzes, and practice exercises completed while studying Google’s Machine Learning Crash Course (MLCC). All notebooks and implementations are organized by module and developed using Google Colab.

About

Google’s Machine Learning Crash Course is a comprehensive introduction to machine learning that combines theory, practical exercises, and real-world applications. This repository serves as a record of my learning journey and showcases my understanding of core machine learning concepts.
